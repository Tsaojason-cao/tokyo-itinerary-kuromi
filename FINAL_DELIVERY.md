## 📅 完整行程概览

- **景点数量**：22个打卡点
- **核心内容**：浅草寺 → 横滨中华街 → 横滨烟火大会 → 晴空塔
- **亮点**：横滨冬季烟火大会（18:00-19:00）

### Day 3 - 2月8日（周日）：富士山一日游
- **状态**：✅ 本次完成

- **核心内容**：富士山五合目 → 忍野八海 → 河口湖

### Day 4 - 2月9日（周一）：镰仓一日游

  - 江之电一日游体验

### Day 5 - 2月10日（周二）：东京市区精华游

- **核心内容**：秋叶原 → 皇居 → 银座 → 筑地 → 东京塔 → 六本木
- **亮点**：
  - 秋叶原动漫文化

  - 筑地海鲜美食
- **门票**：东京塔1,200日元 + 六本木1,800日元

### Day 6 - 2月11日（周三）：Live演出 & 新宿涩谷

## 🎨 设计特色

### 视觉风格
- **主题色**：库洛米粉紫色（pink-purple gradient）
- **设计风格**：卡片式布局，毛玻璃效果
- **图标系统**：丰富的emoji和Lucide图标
- **响应式**：完美适配手机、平板、桌面端

### 交互功能
1. **打卡系统**：每个景点可以勾选打卡，实时显示完成进度
2. **Tab切换**：景点打卡 / 路线导航 / 旅行贴士三个标签页
3. **导航功能**：每个景点都有"导航"按钮（可接入地图API）
4. **进度显示**：顶部显示今日完成度和进度条
5. **天气信息**：每天的天气预报和穿搭建议

### 内容丰富度
每个景点都包含：
- 📜 **历史故事**：景点的历史背景和文化意义
- 💡 **特别说明**：实用提示和注意事项
- 📸 **拍照技巧**：5个专业拍摄建议
- 👩 **女生姿势**：5种适合女生的拍照姿势
- 💑 **情侣姿势**：5种情侣互动拍照姿势
- 🍜 **美食推荐**：当地特色美食推荐
- 🚇 **交通指引**：详细的地铁路线和换乘

## 🚀 如何使用

### 本地开发
```bash
# 克隆仓库
git clone https://github.com/Tsaojason-cao/tokyo-itinerary-kuromi.git
cd tokyo-itinerary-kuromi

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 访问 http://localhost:3000
```

```bash
# 构建生产版本
pnpm build

# 预览生产版本
pnpm preview
```
