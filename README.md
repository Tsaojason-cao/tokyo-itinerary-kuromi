# 东京浪漫之旅 - 库洛米风格行程规划 🌸💜

一个为情侣打造的东京旅行行程规划网页应用，采用可爱的库洛米（Kuromi）粉紫色主题，提供详细的景点信息、拍照姿势指导、美食推荐和地铁导航。

## ✨ 特色功能

### 📍 详细的景点信息
- 20+个精选景点，每个包含历史故事和文化背景
- 特别说明和有趣的冷知识
- GPS坐标和Google Maps一键导航
- 周边美食推荐

### 📸 专业拍照指导
- **女生拍照姿势**（重点）- 每个景点5个详细姿势
- **情侣互动姿势** - 每个景点5个浪漫姿势
- 4-5个专业拍照技巧
- 最佳拍照时间和机位推荐

### 🗺️ 智能路线规划
- 可视化路线图，清晰展示每个步骤
- 详细的地铁换乘指引
- 东京地铁完整线路图
- 步行时间和距离估算

### 💝 情侣友好设计
- 库洛米粉紫色主题
- 浪漫的UI设计
- 适合情侣的拍照姿势
- 打卡进度追踪

### 🌐 实用功能
- 离线可用（PWA）
- 日文语音播报
- 中日英三语支持
- 响应式设计（手机/平板/电脑）

## 📱 使用指南

### 首页
- 查看6天行程概览
- 点击任意天数查看详细行程

### 详细行程页面
1. **景点打卡** - 浏览所有景点，勾选已完成的打卡点
2. **路线导航** - 查看可视化路线图和详细步骤
3. **地铁指引** - 查看地铁线路图和换乘信息

### 拍照功能
- 点击每个景点卡片查看详细信息
- 展开"女生拍照姿势"和"情侣互动姿势"
- 参考拍照技巧获得最佳效果

### 导航功能
- 点击"导航"按钮跳转到Google Maps
- 点击🔊按钮播放日文景点名称

## 🛠️ 技术栈

- **前端**：React 19 + TypeScript
- **样式**：Tailwind CSS 4
- **UI组件**：shadcn/ui
- **路由**：Wouter
- **地图**：Google Maps API
- **构建**：Vite
- **PWA**：Service Worker
